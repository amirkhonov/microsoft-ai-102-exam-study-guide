# AI-102: Designing and Implementing a Microsoft Azure AI Solution

## About This Study Guide

This comprehensive study guide is designed to help you prepare for the **Microsoft AI-102 certification exam**. The AI-102 exam measures your ability to design and implement Azure AI solutions using Azure AI services, Azure AI Search, and Azure Machine Learning.

### Exam Overview

- **Exam Code:** AI-102
- **Exam Title:** Designing and Implementing a Microsoft Azure AI Solution
- **Passing Score:** 700 (on a scale of 1-1000)
- **Duration:** 120 minutes
- **Number of Questions:** 40-60 questions
- **Question Types:** Multiple choice, drag and drop, case studies, and scenario-based questions

### Skills Measured

This study guide is organized according to the official exam domains and their weightings:

1. **Plan and manage an Azure AI solution (20–25%)**
2. **Implement generative AI solutions (15–20%)**
3. **Implement agentic solutions (5–10%)**
4. **Implement computer vision solutions (10–15%)**
5. **Implement natural language processing solutions (15–20%)**
6. **Implement knowledge mining and information extraction solutions (15–20%)**

### How to Use This Guide

1. **Start with PLAN.md** (this file) to understand all exam topics
2. **Navigate to domain folders** (01-06) for detailed documentation and resources
3. **Follow official Microsoft documentation links** provided in each README.md
4. **Practice with hands-on labs** using the quickstarts and tutorials
5. **Review COVERAGE_ANALYSIS.md** to verify complete topic coverage

### Prerequisites

Before taking the AI-102 exam, you should have:
- Experience with Azure services and the Azure portal
- Knowledge of Python, C#, or another programming language
- Understanding of REST APIs and SDKs
- Familiarity with AI concepts and machine learning fundamentals
